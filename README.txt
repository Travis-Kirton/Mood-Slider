**** Instructions to run web application ***

1) This was built using Angular, so ensure you have the following installed
	- NodeJS & NPM (https://nodejs.org/en/download/) LTS is fine

2) Navigate to the root directory and run 'npm install', which should create a Node Modules folder and install the relevent dependencies
	
2) To run the application, navigate in a console to the root directory and run 'ng serve'. Then open Chrome or Firefox and navigate to http://localhost:4200/

3) Tests can be run by using the command 'ng test' from the root directory within a console. Karma will open, displaying test results.

4) XML Document for testing can be found under 'src/assets/xmlDocs'

5) Images used can be found under 'src/assets/images'
